# Office 365 Help Desk Ticketing Simulation

## Overview
This project simulates common Microsoft 365 / Office 365 support tasks typically handled by IT Help Desk, technical support, and entry-level IT operations teams.

## Objective
To demonstrate practical support workflows for Microsoft 365 user issues such as MFA reset and re-registration, shared mailbox access, OneDrive sync issues, Teams sign-in problems, password resets, and basic license or service access verification.

## Included Scenarios
- MFA reset and re-registration support
- Shared mailbox access troubleshooting
- OneDrive sync issue resolution
- Teams sign-in troubleshooting
- Password reset assistance
- Microsoft 365 access and license verification

## Files Included
- `ticket-log.xlsx` — simulated Microsoft 365 help desk ticket log
- `account-support-requests.xlsx` — access and provisioning request tracker
- `mfa-reset-procedure.docx` — MFA reset and re-registration support workflow
- `shared-mailbox-access-report.docx` — shared mailbox access troubleshooting report
